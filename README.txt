1. Работает на gcc и g++. На clang не пробовал, на MSVC не работает.
2. Нужно установить хедеры библиотек. Для этого надо запустить скрипт install_dependencies.sh.
Для его работы нужны некоторые утилиты, которые легко устанавливаются через install_tools.sh
3. После установки может потребоваться перезапустить терминал или прописать source ~/.bashrc
4. Для установки на винде нужно будет открыть PowerShell с правами администратора и
ввести Set-ExecutionPolicy RemoteSigned
